## Run this App

Clone Repo
```
$ git clone https://github.com/codebudy5247/assignment-task.git
```

### Run Backend
```
// In root Dir
$ npm install
$ npm run dev

# Server started on 1337 PORT
```

### Run Frontend
```
$ cd web
$ npm install
$ npm run dev

App Started on http://localhost:5173/
```
